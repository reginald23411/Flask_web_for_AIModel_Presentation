Data Preparation

(A) Sources of data

We used R to select the first 1000 advertisements from a pool of 1500 teacher preparation advertisements.

(B) Data preparation

We used R to handle various issues with the raw data we retrieved.

  1. Missing entries

    2. All missing entries in the raw data were represented by "N/A" or "-". We used the Amelia package and gsub command in R to generate a missingness map for the data. Then, we replaced "N/A" and "-" with "NA" to facilitate further processing in R.

  2. Set up the work database for NLP (Natural Language Processing)

    3. a) We added the titles of the 1000 advertisements to the work database.

    b) We changed the data type of the work database to a vector.

    c) We removed all useless punctuation (e.g., #, /, ()) and digits (0-9) from the titles.

    d) We converted the titles in the work database to lowercase.

  3. Re-formatting

    4. We re-formatted the data by deleting all punctuation (e.g., #) and changing all job codes 0 to "000".

  4. Correcting the data

    5. We used the if-else-if command to correct the job employment types as full-time, part-time, or freelance.
